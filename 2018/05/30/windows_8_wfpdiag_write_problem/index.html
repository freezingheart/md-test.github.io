Windows 8.1系统中wfpdiag.etl 写入量非常大的解决方法

title: Windows 8.1系统中wfpdiag.etl 写入量非常大的解决方法
date: 2015-04-12 17:15:00

tags:

最近下载量比较大，虽然数据都是往机械硬盘写的，但是主硬盘（SSD）写入量还是很大，我的浦科特可禁不起这种折腾，一开始还以为是IDM的锅，然后把IDM的缓存文件路径改为D盘（机械硬盘）以后，情况还是没有好转，打开了资源监视器，调节至磁盘活动 按总数（字节/秒）排序 发现System进程读写C:\Windows\System32\wfp\wfpdiag.etl 的一个文件读写量非常大，排名第一，速百度之，发现这个文件是Windows创建的用于跟踪文件，收集IPsec和防火墙活动的诊断信息。也提供了关闭的方法
Win+X+A （以管理员方式运行命令提示符）同意UAC后，输入

netsh wfp set options netevents = off

返回成功
这时再看资源管理器，SSD的读写占用已经明显低下来了。
