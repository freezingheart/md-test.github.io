小白向的系统重装教程

title: 小白向的系统重装教程
date: 2014-05-09 18:33:00

tags:

说明
==

很多新手在重装系统的时候都要找所谓的“高手”来帮忙重装系统，在大家眼中重装系统好像是件很高科技的事，其实重装系统很简单，从 Windows 的最初版本开始微软就在不停的改进系统安装的方法，每一代的系统比起上一代来说都更容易安装 / 使用。所以不用担心重装系统会很麻烦，你要做的就是把你电脑里你认为重要的东西、不想删掉的东西，复制到其他地方避免丢失，然后就可以跟着教程重装系统了。
在这个教程中，我会用尽可能容易理解的方式来教大家怎样重新安装操作系统。
我会尽可能把应该考虑的细节写下来，虽然可能有点啰嗦，不过很多东西只要大概了解下就好。

为什么要自己重装系统？
===========

电脑用了一段时间之后，常常会因为各种问题而恼火：功能不正常、操作卡顿、蓝屏死机、中了木马病毒……这些问题大多都能通过重装系统得到解决。
学会怎样重装系统之后，当你电脑崩溃的时候就不再需要到处求人了。而且现在电脑 90% 的问题都能通过重装系统解决，如果重装解决不了，那便很可能需要更换电脑中的零部件了。
自己重装的系统可以放心，这样的系统完整又干净，按教程说的装好系统也需要担心病毒木马。
另外还可以在女朋友 / 男朋友面前炫耀一把

决定好你要安装什么版本的系统
==============

先想好你要装什么系统，不然之后后悔是件麻烦的事。
建议的系统有：

> Windows 7 （推荐，发布时间较旧）
> Windows 8 （前两年发布的系统，采用了很多新技术，速度快，有的软件可能不能用）
> Windows 8.1 （是 Windows 8 的更新版本，更适合用鼠标操作）

决定之后你需要下载安装镜像，一个比较靠谱的下载地址是：http://msdn.itellyou.cn 例如需要安装 Windows 7，则在左边依次点击 操作系统 -> Windows 7 -> 简体中文 如果你有正版的序列号，可以选择序列号对应的系统； 如果你不是正版用户，那么就选择一个功能全一点的。


个人建议的选择方案：

Windows 7 Ultimate with Service Pack 1（旗舰版，带SP1，可以少打很多补丁）
Windows 8（基本版和专业版都有，安装时建议安装专业版）
Windows 8.1 （同上，请下载 Multiple Edition 合盘版）

你会发现同样的系统有 x86（32位）和 x64（64位）两个版本，怎么选择呢？
我的建议是：如果你的电脑是好几年前的，或者内存小于 2GB，那么可以考虑 x86，否则请选择 x64.
点开“详细信息”之后你就能看到一个 ed2k:// 开头的地址，把他复制到你的迅雷就可以开始下载了。

下载好之后，我们就能用这个安装镜像开始安装系统了。

制作安装源

常见的安装办法有两种，一种是 U 盘安装，另一种是光盘安装。
光盘安装很简单，你只需要通过 UltraISO 或者其他工具把它刻录到光盘上就能开始安装。
不过我个人更推荐 U 盘安装的方法，因为现在 U 盘速度远超过光盘，如果你有一个高质量的 USB 3.0 U盘（个人推荐U盘 Sandisk CZ80），重装系统也就是泡一碗泡面就能搞定的事情。

制作安装U盘

第一步，把 U 盘格式化成 FAT32 格式。（如果U盘是FAT32格式可以不用格式化）
PS:如果是Legacy Bios 可以不用FAT32 格式（FAT32格式大小有局限，单文件最大4G），可以使用exFAT格式（专为U盘准备）但是别用NTFS格式！！

扩展阅读：为什么要用 FAT32 格式？
这里涉及到近几年的新技术 —— UEFI (统一可扩展固件接口)
在 UEFI 启用的情况下，安装系统必须使用 FAT32 格式的 U 盘才能够进入安装程序；
而且安装系统时必须用新的 GPT 分区格式。（如果之前是MBR格式的推荐重新分区）

为何U盘一定别用NTFS格式？
我们看近几年的硬盘格式都是NTFS格式了，这种格式有许多优点，例如可靠的文件系统，支持文件夹压缩，对磁盘空间的有效管理等，但是由于第一个优点（可靠的文件系统），所以NTFS采用日志制的读写记录方式，需要大量的读写闪存，由于U盘 这种存储的特殊性，其有读写次数限制，到一定就无法写入了，所以说这种格式比较伤U盘。

第二步，先把镜像解压到U盘里，这里不再介绍。
第三步，下载、解压并运行这个软件 BOOTICEx86.zip
选择你的 U 盘，点击“主引导记录”
选择 Windows NT 5.x / 6.x MBR，点击安装，然后关闭“主引导记录”窗口
选择“分区引导记录”，选择 BOOTMGR 引导程序，点击安装
关闭 BOOTICE。
这样你的 U 盘就能够用来安装系统了。

进入安装程序

重新启动电脑，按键选择启动方式。（ESC、F12、F11、F10）

问：魂淡！你这么说我鬼知道怎么按啊！！！
答：很简单啊，仔细看开机前几秒钟屏幕左下角的提示，一般会有 Press ** key to enter boot menu（不同电脑可能有所不同）
如果没有这个菜单，也会有按键提示，有的按键后会进入蓝色的 BIOS 界面，在 BIOS 中也可以修改开机顺序。具体请看其它教程或者是主板说明书（建议有一定计算机基础的阅读主板说明书）。
另外 UEFI 用户在选择过程中会看到“UEFI: **”的启动方式，选择此项就能在UEFI基础上安装操作系统了（比传统方式运行更快）
光盘用户会看到一行“Press any key to boot from CD / DVD”，在提示信息显示过程中按下任意键就能进入安装程序。
如果看到一行 Windows is loading file，同时下面有个进度条在慢慢加载，恭喜你，你已经进入了 Windows 的安装程序。

安装操作系统

等待上一步加载好安装程序，就会看到 Windows 7 / 8 的欢迎界面，用鼠标点击“下一步”，点击“立即安装”
稍等片刻就会进入输入序列号的界面，盗版用户请自行搜索“Windows * 密钥”。
部分盗版用户输入后还需要选择操作系统版本，按之前章节说的选择
序列号分两种：Retail（零售版） 和 VOL（批量授权版），如果提示不对请换另一种类型的试试。
接下来到“软件许可协议”，选择“我接受许可条款”，点击下一步
到了安装方式选择界面，请务必选择“自定义”，升级安装慢而且也不靠谱。
接下来系统要求选择要安装的磁盘，也就是C盘是哪个。

强烈建议安装前格式化 C 盘，原来的系统和程序基本又用不了，还浪费空间
格式化方法：(右侧)驱动器选项 -> 格式化 -> 确定
PS：格式化前看清楚大小和卷标，手滑格错了这个真的别怪我 = =
选择后点击下一步就能开始安装了，安装过程是全自动的，对，你没有看错，全自动的。
你现在可以去泡杯咖啡吃点零食了，电脑会自动帮你搞定。

有的电脑会提示无法安装到此分区，原因一般有这几个：
空间不够（难道你选错磁盘了？）
UEFI安装，磁盘却又是MBR格式的（参考上面UEFI介绍，要么重新分区，要么开机选择过程中不要选择带UEFI的）
传统安装方式，磁盘却又是GPT格式的（这个……你跟Case 2的结婚去吧 = =）
完成安装

完成安装后一般系统会显示一个对话框，要你输入计算机名等信息。
这个不多介绍，按照提示输入信息就好，错了也问题不大。
完成配置之后，就会进入系统了。接下来就是安装你常用的软件了。
软件安装不多说，这里特别提下杀毒软件，如果你是小白的话个人比较推荐这些：

金山毒霸 （留意不要安装卫士、浏览器和其他，那些没用的）
QQ电脑管家 （同上，小心捆绑）
国外杀毒软件 （强烈推荐）
360 卫士（你确定这货不是病毒？）
PS：上面这些装一个就好，别我列出来你全装了……
（国产杀软都没节操，装太多会卡的哦~）
