配置本地yum更新源

title: 配置本地yum更新源
date: 2015-02-03 09:54:00

tags:

重要！！！挂载的光驱在重启后会自动解挂 在重启后重新挂载或者复制到指定文件夹！！！
1. 首先 建立/media/CentOS 文件夹（默认的配置文件内就用的此文件夹 这样可以少一些麻烦）
mkdir /media/CentOS


2. 挂载光盘 挂载点为刚刚创建的/media/CentOS文件夹
mount /dev/cdrom /media/CentOS

3. 更改目录至/etc/yum.repos.d/
cd /etc/yum.repos.d/


  1. 将默认的CentOS-Base.repo文件更改文件名为CentOS-Base.repo.bak作为备份
    mv CentOS-Base.repo CentOS-Base.repo.bak

    5.编辑CentOS-Media.repo文件，此文件为通过本地yum途径获取rpm包的重要步骤
    vim CentOS-Media.repo

    修改 20行enabled=1
    :wq退出
  2. 将之前的yum缓存清除
    yum clean all

    7.yum list 查看源
